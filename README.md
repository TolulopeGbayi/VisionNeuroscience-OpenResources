# VisionNeuroscience-OpenResources
Open-source Python notebooks, notes, and reproducible workflows to be derived from the Virtual Systems Vision Science Summer School and my symposium project

This repository contains Python notebooks, notes, and reproducible workflows earnt and developed from the pre-course materials and Course of the Virtual Systems Vision Science Summer School (https://zhaoping.thinkific.com/collections) and my symposium project presentation. 

These resources are shared under the MIT License to promote open science and accessible computational neuroscience research.
